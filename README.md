# Titanic Exploratory Data Analysis (EDA)

## Project Overview

This project performs exploratory data analysis (EDA) on the Titanic dataset using Python and popular data analysis libraries.

The objective is to identify patterns and factors that influenced passenger survival on the Titanic while demonstrating clean and reproducible data analysis practices.

---

## Objectives

- Load and inspect the Titanic dataset
- Clean missing and inconsistent data
- Explore relationships between features and survival
- Visualize important trends
- Summarize findings clearly

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
